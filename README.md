# My Life Organizer

A beautiful, interactive dashboard to organize and track all aspects of your life including career goals, learning projects, financial goals, personal development, work projects, and life balance.

## Features

- **Career Goals Tracking** - Monitor your professional development and career transitions
- **Learning Projects** - Keep track of educational pursuits and skill-building projects
- **Financial Goals** - Manage your financial objectives and track progress
- **Personal Development** - Document personal growth initiatives
- **Work Projects** - Track current work assignments and certifications
- **Life Balance** - Monitor health, relationships, and hobbies

## Key Functionality

- Add, edit, and delete items across all categories
- Priority system (High, Medium, Low) with color-coded indicators
- Status tracking (Active, Planning, Completed)
- Interactive progress bars (0-100%)
- Timeline management for goal deadlines
- LocalStorage persistence - all data saved automatically
- Export/Import functionality for data backup and transfer
- Fully responsive design for mobile and desktop
- No server required - works completely offline

## How to Use

1. Open `life_organizer.html` in any modern web browser
2. Click the "+ Add" buttons to create new items in any category
3. Use the Edit button (yellow) to modify existing items
4. Use the Delete button (red) to remove items
5. Click on progress bars to update completion percentages
6. Export your data regularly for backup
7. Import previously exported data to restore

## Technologies Used

- HTML5
- CSS3 (with animations and gradients)
- Vanilla JavaScript
- LocalStorage API

## Data Persistence

All your data is stored locally in your browser's localStorage. To backup your data:
1. Click the "Export Data" button to download a JSON file
2. Save this file in a secure location
3. Use "Import Data" to restore from a backup
